# Category 5: Data Analysis

## Project Overview
Cleaned a synthetic messy customer support tickets dataset and got some insights on resolution times.

## Files
- `synthetic_customer_support_tickets_messy.csv` - Original messy data
- `category5_data_analysis.ipynb` - Full analysis notebook
- `resolution_time_analysis.png` - Graphs

## How to Run
1. Open `category5_data_analysis.ipynb` in Jupyter Notebook
2. Run all cells

## Key Findings
- Average resolution time: 122.47 hours
- Slowest category: Data Sync Issue
